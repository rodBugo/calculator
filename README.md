# Calculator

## Overview

This project is a browser-based calculator application built using HTML, CSS, and JavaScript. The goal of this project was to strengthen my understanding of JavaScript fundamentals, DOM manipulation, event handling, and application state management.

The calculator supports both mouse and keyboard input, allowing users to perform basic arithmetic operations through an interactive interface.

## How It Works

* Users can input numbers using either on-screen buttons or keyboard keys.
* The calculator supports addition, subtraction, multiplication, and division.
* Expressions are displayed in real time as users enter values and operators.
* Results are rounded to prevent long floating-point decimals.
* Division by zero is safely handled with an error message.
* Users can clear the calculator or remove individual digits using backspace functionality.
* Calculations can be chained together for continuous operations.

## Features

* Basic arithmetic operations
* Real-time expression display
* Keyboard support
* Backspace/delete functionality
* Clear/reset functionality
* Decimal number support
* Division-by-zero error handling
* Rounded floating-point results
* Chained calculations
* Interactive browser-based UI

## How to Run

* Clone or download the repository
* Open the project folder
* Open `index.html` in your browser
* Use the calculator buttons or keyboard to enter calculations
* Press `=` or `Enter` to evaluate expressions
* Press `Backspace` to remove digits
* Press `Escape` or the clear button to reset the calculator
